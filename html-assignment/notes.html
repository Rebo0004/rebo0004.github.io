HTML tags

nav
Contain links to other pages of the website.
head
It's the brain of the html document. It's mainly for the browsers, contains links and information.
section
It groups related content on the page. Must have its own heading.
form
It's used to collect various types of data. To make it functional needs to be connected with backend or server-side languages like PHP or Python
select
Organized elements in a dropdown container.
